⚽ FIFA WORLD CUP ANALYTICS DASHBOARD

🔷 📊 OVERVIEW : 
This project is an interactive dashboard built using Power BI to analyze historical FIFA World Cup data.
It provides insights into matches, teams, goals, and tournament trends from 1930 to 2022.
___________________________________________________________________________________________________________________________________________________________________

🔷 🎯 OBJECTIVE :
Analyze World Cup data
Identify key trends and patterns
Present insights using visual dashboards
___________________________________________________________________________________________________________________________________________________________________

🔷 📂 DATASET :
Source: Kaggle
Coverage: 1930 – 2022
🔷 📈 KEY HIGHLIGHTS
⚽ Total Matches, Goals, and Teams
🏆 Winner & Performance Analysis
🌍 Country-wise Participation
📅 Year-wise Trends

___________________________________________________________________________________________________________________________________________________________________

🔷 🛠️ TOOLS USED :
Power BI
Data Cleaning & Visualization
___________________________________________________________________________________________________________________________________________________________________

🔷 🚀 HOW TO USE :
Download the .pbix file
Open it in Microsoft Power BI Desktop
Explore dashboard pages
